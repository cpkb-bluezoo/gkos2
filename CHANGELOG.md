# Changelog

## 1.0.0 — 2026-02-14

Initial release.

- 6-key chorded keyboard with swipe-to-chord input
- 18 language layouts: English, French, German, Dutch, Spanish, Portuguese,
  Italian, Danish, Norwegian, Swedish, Finnish, Greek, Russian, Korean,
  Esperanto, Estonian, Icelandic, Ukrainian
- Dual layout variants (GKOS Standard and frequency-optimized) for languages
  with an official GKOS layout; optimized-only for Dutch
- Predictive text suggestions from bundled word frequency lists (50K words per
  language) and a personal user dictionary that learns from your typing
- Emoji picker (via AndroidX emoji2)
- Unicode hex input mode for entering any Unicode character by codepoint
- Progressive disclosure: primary outcomes on keys, secondary outcomes around
  keys for chord discovery
- Mode indicator for Shift, NUM, and SYMB states
- Dark and light mode support
- Settings activity for language and variant selection
- No network permissions — all data stays on device
