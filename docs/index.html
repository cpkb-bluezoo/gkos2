GKOS2 Chord Reference

Visual chord diagrams for every supported keyboard layout

About GKOS

GKOS (Global Keyboard Open Standard) is a chorded keyboard system that uses just six keys — three on each side of the screen — to produce the full range of letters, numbers, symbols, and control actions. Each key can be pressed individually, or multiple keys can be combined into a chord to select a different character.

On a touchscreen you don't need to press multiple keys simultaneously. Instead, you can swipe across adjacent keys in a single gesture: every key your finger passes through is added to the chord, and the character is emitted when you lift your finger. This makes one-handed typing fast and comfortable.

GKOS2 is an Android keyboard app that implements this system with 18 language layouts, predictive text, an emoji picker, and Unicode hex input. It requires no network permissions — all data stays on your device.

How to Read the Diagrams

Each diagram shows every chord combination for a given language layout. The six physical keys are labelled A–F: A, B, C on the left and D, E, F on the right, from top to bottom.

key = pressed (inverted, white-on-black)   key = unpressed

Centre (large keys)
Single-key outcomes in ABC mode, shown large on the keys themselves.
Gray text
The outcome in ABC123 / NUM mode (e.g. digits, punctuation).
Blue text
The outcome in SYMB mode (e.g. symbols, accented characters).
Inner chord cells
Two-key chords inside the main box (AB, BC, DE, EF, AC, DF). The character appears as a dark badge between the pressed keys; three-key extensions appear on the opposite-side keys. Gray and blue annotations show NUM/SYMB outcomes.
Outer chord cells
Cross-side and multi-key chords (AE, ABEF, BCDE, etc.) shown in the panels flanking the main box.
Navigation & Actions
Arrow keys, page up/down, backspace, and space.
Mode Switches
SHIFT, SYMB, and ABC⇄123 mode toggles.
5-Key Control Chords
Esc, Ctrl, Alt, Enter, Tab, and Delete.

Most layouts come in two variants: Standard (the official GKOS letter placement) and Optimized (letters reordered by frequency for faster typing in that language).
